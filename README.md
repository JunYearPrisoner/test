# 瞬间的花火 

    戒骄戒躁，
    戒无度用脑，
    人生唯有依靠耐心与毅力。
    世间最怕的，
    便是耐心与毅力这二者。
    火花虽美，
    却只留一瞬记忆。
    
             -------Natsume Soseki 
             
    
    风吹走了雨，
    也吹散了工人的歌谣和他的思绪。
    他没有点燃卷烟，
    他感受到了，
    一种近乎欢愉的苦楚。

             -------Ryunosuke AKutagawa
